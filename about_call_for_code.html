About Call For Code

by Nwamu Japhlet Chinonso

Posted on June 12, 2019

Hello Everyone! As promised earlier, we'll be giving you all a brief intro into the IBM #CallforCode.

ABOUT THE IBM CALLFORCODE 2019
The Call for Code 2019 Global Challenge is a worldwide developer competition that seeks technology solutions for natural disaster preparedness, response, and recovery. It is supported by the IBM Code and Response™ initiative, a multi-year program dedicated to creating and deploying open source technologies to tackle the world's biggest challenges. The winning team receives $200,000 USD, support from The Linux Foundation, implementation through Code and Response, and more. Do you have big ideas that can create big changes? Then this is your chance to have an impact on the world! You're good at Data Science? Create algorithms that understand, analyze, and predict health and nutrition needs to improve services with data science. You're good at AI? Use AI and bots to improve real-time communications with natural language processing. Know Blockchain? Then build secure, resilient, traceable, and transparent supply networks with blockchain.

"The winning team receives $200,000 USD, support from The Linux Foundation, implementation through Code and Response, and more."

According to #callforcode

You could also write algorithms to improve logistics based on traffic and weather activity to reduce the number of people affected. For those good at Internet of things, collect and analyze device sensor data to take corrective or preventative action automatically. You're good at Machine Learning? Then use machine learning, deep learning, and visual recognition to improve critical processes. Love to join the #CallforCode challenge? Then, * Join the Call for Code 2019 community here: https://cloud.ibm.com/registration… * Build your solutions with using Code Patterns in these six tech areas mentioned above, six disaster areas, and health to inspire your submission. * Engage experts and find teammates. * Then submit and iterate on your solution. Today, Cregitech commits to IBM's #CallforCode challenge. And as #CallforCode partners, we are here to help you through the whole process if you're having any difficulties. Can't join the #CallforCode challenge because you don't have any of the skills mentioned above? Don't worry, next year gives you another chance to participate! Soon we'll be partnering with IBM to train interested developers the Cloud, AI, Data and Blockchain technologies.
